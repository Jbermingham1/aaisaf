# ISO/IEC 42001 Mapping

Maps AAISAF techniques to ISO/IEC 42001:2023 (Artificial Intelligence Management System) requirements.

| ISO 42001 Clause | AAISAF Coverage |
|:-----------------|:----------------|
| **4.1** Context of the organisation | Threat landscape (02-threat-landscape.md) |
| **6.1** Actions to address risks | Full taxonomy (TA01–TA10), AISS scoring |
| **6.2** AI objectives | Maturity model (5 levels) |
| **8.1** Operational planning | Assessment methodology, system classification |
| **8.4** AI system impact assessment | AISS scoring, compliance mappings |
| **A.5.4** AI system documentation | Technique documentation standard |
| **A.6.2.4** Input validation and filtering | TA01 (prompt), TA06.001 (voice), TA10.010 (args) |
| **A.6.2.5** AI system robustness | TA01, TA06.004, TA08.001 |
| **A.6.2.6** Third-party AI components | TA05 (supply chain), TA10 (MCP tools) |
| **A.8.5** Data for AI systems | TA02 (RAG), TA07 (training data) |
| **A.9.3** AI system monitoring | Detection guidance in all 87 techniques |
