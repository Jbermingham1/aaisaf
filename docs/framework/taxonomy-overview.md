# Taxonomy Overview

AAISAF uses a three-level taxonomy modelled on MITRE ATT&CK:

```
Tactic (TA01–TA10)
  └── Technique (TA01.001–TA01.012)
        └── Sub-technique (where applicable)
```

| Tactic | Techniques | Critical | High | Medium | Low |

### Novel Tactics

**TA06 (Voice AI)** and **TA10 (MCP & Tool Security)** are globally first — no other framework provides technique-level coverage for these attack surfaces.
